Soluție personalizată

Soluție personalizată în funcție de nevoile fiecăruia.

myImmoAssistant vine în patru distribuții: cloud public, whitelabel, cloud privat/hibrid sau direct instalat pe server propriu (în datacenter propriu).

  • Contact

Cloud public

Soluția distribuită într-un cloud public AWS/Azure/GCP.

Whitelabel

Soluția personalizată pe baza cerințelor/nevoilor clientului.

Cloud privat/hibrid

Soluția actuală sau cea personalizată poate fi distribuită în cloud-ul privat/hibrid al clientului.

Datacenter

Soluția actuală sau cea personalizată poate fi distribuită în datacenter-ul clientului.
